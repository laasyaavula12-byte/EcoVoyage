# EcoVoyage

EcoVoyage is a responsive travel website created as part of the MANTRA 2026 Summer School Frontend Website Assignment.

## Topic
Travel Website

## Features
- Responsive design for desktop, tablet, and mobile
- Home, About, Discover, Gallery, and Plan Journey pages
- Destination search
- Dark mode
- Image gallery
- Animated counters
- Travel facts
- Journey planning form

## Technologies Used
- HTML5
- CSS3
- JavaScript
